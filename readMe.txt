Project Name: My Advanced Blog
Project Explanation:
It consists of a header, content and footer.
Header: It contains the title and some links to facilitate access to other pages.
Content: It consists of two sections, the first section consists of 3 displayed posts and there is a button to read each post separately. As for the second section, it is information about the blog and the same links found in the navigation bar and links to some recent links.
Footer: It consists of a contact form so that the user can send messages and copy right.

Running the project: The project can be run by double-clicking to open a file named index. A new tab will open in the browser to view the project.
